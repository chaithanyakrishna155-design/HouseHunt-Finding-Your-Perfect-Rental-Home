# 🏠 HouseHunt – Finding Your Perfect Rental Home

A full-stack web application that simplifies the process of finding, listing, and managing rental properties. The platform provides separate dashboards for Renters, Property Owners, and Administrators, enabling secure property management, booking, and user administration.

---

## 📌 Project Overview

HouseHunt is a House Rental Management System developed as part of the **Deloitte Capstone Program 2026**.

The application enables:

- 🏡 Renters to search and book rental properties
- 🏠 Property Owners to list and manage their properties
- 👨‍💼 Administrators to manage users, properties, and bookings

# ⚙️ Installation

## Clone the repository

```bash
git clone https://github.com/chaithanyakrishna155-design/HouseHunt-Finding-Your-Perfect-Rental-Home.git
```

Move into the project

```bash
cd HouseHunt-Finding-Your-Perfect-Rental-Home
```

---

## Backend Setup

```bash
cd "Project files/backend"
npm install
npm start
```

Backend runs on:

```
http://localhost:8001
```

---

## Frontend Setup

```bash
cd "Project files/frontend"
npm install
npm start
```

Frontend runs on:

```
http://localhost:3000
```

---

# 🔐 Environment Variables

Create a `.env` file inside:

```
Project files/backend
```

Example:

```env
MONGO_URL=your_mongodb_connection_string
JWT_KEY=your_secret_key
```

> **Note:** The actual `.env` file is intentionally excluded from this repository for security purposes.
